## Event Booking web application 
#### GraphQL, MongoDb, Express.js, React, Nodejs, Jsonwebtoken

#### Quick start 
`npm i -g`
`cd client`
`npm i`
#### For server testing localhost:5000
`npm run server`
#### For whole application testing localhost:3000
`npm run dev`
